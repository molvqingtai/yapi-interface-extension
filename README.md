# yapi-interface-extension

一个 YAPI 浏览器插件，在 YAPI 接口文档显示 TypeScript interface。



### Install

将根目录 **yapi-interface-extension.zip** 解压后的 **dist** 文件夹拖入 Chrome 扩展界面安装。
